# Math and CS

- This is a [scala-cli](https://scala-cli.virtuslab.org/) project.

- There are two "projects" (due to binary incompatibility) in two directories:
  - `axle` (using Scala 2.13.13) and
  - `core` (using Scala 3.4.0).

- You can open these folders separately in VS Code,

- or run them separately from the root of the main folder as:

```scala
scala-cli test ./axle
scala-cli test ./core
```
